LIRI Bot

My Roll

I built this project alone. The goal for this project was to create a command line app using node.js. The app takes in various commands along with a search parameter and gives back data based on the command.

Challenges

The challenge in this project was calling multiple APIs through the command line. Another challenge was keeping the Spotify API key private.

Solutions

I broke the API calls into individual functions that would run based on the command entered. In order for the app to function correctly, it was crucial that I installed the correct node packages and included the necessary requirements in the main javascript file. To keep the Spotify API keys private, I included the keys in a .env file and used the dotenv package to call it.

Technologies
  • Javascript
  • Node.js
  • Node-Spotify-API
  • Bands In Town API
  • OMDB API
  • Axios
  • Moment.js
  • DotEnv
